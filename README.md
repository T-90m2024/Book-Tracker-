# Book-Tracker-Трекер прочитанных книг

**Автор:** Чижов Юрий

**Дата:** 04.05.2026

## Описание программы

GUI-приложение для управления списком прочитанных книг.  
Позволяет добавлять книги, фильтровать по жанру и количеству страниц, удалять записи.  
Данные автоматически сохраняются в файл `books.json`.

## Требования

- Python 3.6+
- Стандартные библиотеки: tkinter, json, os

## Инструкция по запуску

1. Скачайте файл `book_tracker.py`
2. Откройте терминал в папке с файлом
3. Запустите программу:
   ```bash
   python book_tracker.py
   Как пользоваться
Добавление книги
Заполните поля и нажмите «Добавить книгу». Проверяется, что все поля заполнены и количество страниц — число.

Фильтрация

Введите жанр (частичное совпадение, регистронезависимо)

Введите минимальное количество страниц (например, 200)

Нажмите «Применить фильтр»

Кнопка «Сбросить фильтр» показывает все книги

Удаление
Выберите книгу в таблице и нажмите «Удалить выбранную книгу»

Сохранение
Автоматически при каждом добавлении/удалении. Файл books.json создаётся в той же папке.

Примеры использования
Пример 1: Добавление книги
text
Название: 1984
Автор: Джордж Оруэлл
Жанр: Антиутопия
Страниц: 328
→ Книга добавлена в таблицу.
Пример 2: Фильтр по страницам
Фильтр: Страниц больше: 300
→ Покажет книги с количеством страниц > 300, например «1984» (328 стр.).

Пример 3: Фильтр по жанру
Жанр: фантастика
→ Покажет книги, в жанре которых есть слово «фантастика».

Тестирование (чеклист)
Пустые поля → ошибка

Страницы буквами → ошибка

Добавление книги → появляется в таблице

Фильтр работает корректно

Сброс фильтра возвращает все книги

Удаление книги → исчезает из таблицы

После перезапуска данные сохраняются (проверьте books.json)
